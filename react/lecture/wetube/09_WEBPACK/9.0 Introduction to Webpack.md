# Explanations


WebPack은 내가 작성한 Javascript 코드를  Browser에 탑재하기 위해  용량이 줄어든 코드로 압축 시켜준다.

특정 플러그인을 설치하면 최신 Javascript 코드를 
Browser가 이해할 수 있는 구 버전 코드로 변경 시키는 역할도 한다.

babel-node가 최신 Javascript 코드를 
node가 이해할 수 있는 코드로 변경해주는것과 비슷하다.
